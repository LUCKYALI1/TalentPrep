 # TalentPrep

An AI-powered interview preparation platform built to help candidates
practice interviews, improve communication, and track progress through
an intuitive, modern web interface.

## ✨ Features

-   🎤 AI-powered mock interviews
-   📊 Performance dashboard
-   📝 Interview history
-   🌙 Light/Dark theme
-   🔐 Authentication
-   📱 Responsive UI
-   ⚡ Fast React + Vite frontend
-   🗄️ Node.js + Express + MongoDB backend

## 🛠️ Tech Stack

### Frontend

-   React.js
-   Vite
-   Tailwind CSS
-   Axios
-   React Router
-   Context API

### Backend

-   Node.js
-   Express.js
-   MongoDB
-   Mongoose
-   JWT Authentication

## 🚀 Getting Started

### Clone the repository

``` bash
git clone <repository-url>
cd TalentPrep
```

### Install dependencies

Frontend

``` bash
cd client
npm install
```

Backend

``` bash
cd ../server
npm install
```

### Environment Variables

Create a `.env` file inside the `server` directory.

``` env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
```

### Run the application

Backend

``` bash
npm run dev
```

Frontend

``` bash
cd client
npm run dev
```
